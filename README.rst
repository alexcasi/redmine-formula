=======
Redmine
=======

Install and configure Redmine.

Available states
================

.. contents::
    :local:

``redmine``
-----------

Runs the states to install and configure redmine.

``redmine.install``
-----------

Installs Redmine from Subversion.

``redmine.service``
-----------

Register Redmine as a service.

``redmine.unicorn``
-----------

Install unicorn script for redmine.
